## はじめに

本マニュアルは、各種休暇の申請方法について説明するものです。適切な休暇取得により、ワークライフバランスを実現しましょう。

### 基本ルール

- **事前申請が原則**: 緊急時を除き、事前に申請・承認を得ること
- **計画的な取得**: 業務スケジュールを考慮し、計画的に取得すること
- **引継ぎの徹底**: 休暇中の業務は適切に引き継ぐこと
- **緊急連絡先**: 長期休暇時は緊急連絡先を上長に伝えること

## 年次有給休暇

### 付与日数

| 勤続年数 | 付与日数 |
|---------|---------|
| 6ヶ月 | 10日 |
| 1年6ヶ月 | 11日 |
| 2年6ヶ月 | 12日 |
| 3年6ヶ月 | 14日 |
| 4年6ヶ月 | 16日 |
| 5年6ヶ月 | 18日 |
| 6年6ヶ月以上 | 20日 |

### 取得単位

1. **1日単位**: 所定労働時間(8時間)を休む
2. **半日単位**: 午前(9:00〜13:00)または午後(13:00〜18:00)を休む
3. **時間単位**: 1時間単位で取得可能(年5日まで)

### 申請方法

#### 1日・半日単位の場合

**申請期限**: 原則として前日まで

**手順**:
1. 社内ポータル「勤怠管理」→「有給休暇申請」を選択
2. 休暇日、取得単位を入力
3. 理由を簡潔に記入(詳細不要、「私用のため」等でOK)
4. 「申請」ボタンをクリック
5. 上長がシステム上で承認

**注意点**:
- 繁忙期や重要な会議がある日は避ける
- チームメンバーと調整し、業務に支障がないようにする

#### 時間単位有給休暇の場合

**申請期限**: 前日まで

**取得可能時間**:
- 1時間単位で取得
- 年間5日(40時間)まで

**利用例**:
- 通院: 10:00〜12:00の2時間
- 役所手続き: 14:00〜16:00の2時間
- 子供の学校行事参加: 15:00〜17:00の2時間

**手順**:
1. 社内ポータル「勤怠管理」→「時間単位有給申請」を選択
2. 日付、開始時刻、終了時刻を入力
3. 理由を記入
4. 「申請」ボタンをクリック

### 有給休暇の繰越

- 未使用分は翌年度に限り繰り越し可能
- 2年間で消滅する
- 例: 2024年4月付与分は、2026年3月末で失効

### 計画的付与

会社は年5日を限度に、計画的に有給休暇を付与することがあります。

**2024年度の計画付与日**:
- 8月14日(水): 夏季休暇として全社一斉付与
- 12月30日(月)、31日(火): 年末年始休暇として全社一斉付与

### 年5日の取得義務

法律により、年10日以上の有給休暇が付与される従業員は、年5日以上取得する義務があります。

**対象期間**: 有給休暇の付与日から1年間

**未消化の場合**:
- 9月時点で取得日数が少ない場合、人事部から通知
- 会社が時季を指定して取得させることがある

## 特別休暇

### 慶弔休暇(有給)

結婚、出産、家族の死亡等の慶弔事由が発生した場合に取得できます。

#### 慶事

| 事由 | 日数 | 取得期限 |
|-----|------|---------|
| 本人の結婚 | 5日 | 結婚日から6ヶ月以内 |
| 配偶者の出産 | 3日 | 出産予定日の2週間前〜出産後2週間 |
| 子の結婚 | 1日 | 結婚式当日を含む前後1週間 |

#### 弔事

| 事由 | 日数 | 取得期限 |
|-----|------|---------|
| 配偶者の死亡 | 5日 | 死亡日から1ヶ月以内 |
| 父母の死亡 | 5日 | 死亡日から1ヶ月以内 |
| 子の死亡 | 3日 | 死亡日から1ヶ月以内 |
| 兄弟姉妹の死亡 | 3日 | 死亡日から1ヶ月以内 |
| 祖父母の死亡 | 1日 | 死亡日から1ヶ月以内 |
| 配偶者の父母の死亡 | 3日 | 死亡日から1ヶ月以内 |

**申請方法**:
1. 社内ポータル「勤怠管理」→「特別休暇申請」
2. 事由を選択
3. 該当する日付を入力
4. 証明書類(婚姻届受理証明書、死亡診断書等)を添付
5. 緊急の場合は電話・メールで事前連絡し、事後に正式申請

### 夏季休暇(有給)

**付与日数**: 3日
**取得期間**: 7月1日〜9月30日
**取得方法**: 年次有給休暇と同様に申請

**推奨取得パターン**:
- お盆休み(8月13日〜15日)と組み合わせて連休化
- 3日を分散して取得も可

### リフレッシュ休暇(有給)

長期勤続者に対する慰労休暇です。

| 勤続年数 | 日数 |
|---------|------|
| 5年 | 3日 |
| 10年 | 5日 |
| 15年 | 5日 |
| 20年 | 7日 |
| 25年 | 7日 |
| 30年 | 7日 |

**取得期間**: 該当年度の4月〜翌年3月
**推奨**: 連続取得でリフレッシュ

### ボランティア休暇(有給)

災害救援、社会貢献活動に参加する場合に取得できます。

**付与日数**: 年3日まで
**対象活動**:
- 災害ボランティア(被災地支援)
- NPO・NGO活動
- 献血、骨髄ドナー

**申請方法**:
1. 事前に活動内容を上長・人事部に報告
2. 活動後、参加証明書を提出

## 病気休暇

私傷病により欠勤する場合に取得できる有給休暇です。

**付与日数**: 年10日
**対象**: 本人の病気・ケガ
**給与**: 100%支給

### 申請方法

#### 短期(1〜2日)の場合

1. 当日の始業時刻までに上長に電話・メールで連絡
2. 復帰後、社内ポータルで申請
3. 診断書は不要

#### 長期(3日以上)の場合

1. 当日の始業時刻までに上長に電話・メールで連絡
2. 医師の診断書を取得
3. 復帰後、診断書を添付して申請

### 病気休暇を超える欠勤

病気休暇(年10日)を使い切った場合:
1. 年次有給休暇を使用
2. 有給休暇も使い切った場合は欠勤扱い(無給)
3. 長期療養が必要な場合は「休職制度」を検討

## 産前産後休業

### 産前休業

**取得期間**: 出産予定日の6週間前(多胎妊娠は14週間前)から
**取得方法**: 本人の請求による

### 産後休業

**取得期間**: 出産日の翌日から8週間
**強制休業**: 産後6週間は就業禁止
**例外**: 産後6週間経過後、本人が請求し医師が認めた業務は就業可

### 給付金

**出産育児一時金**: 50万円(健康保険から支給)
**出産手当金**: 標準報酬日額の2/3(健康保険から支給、産前42日+産後56日分)

### 申請方法

1. 妊娠が判明したら、速やかに上長・人事部に報告
2. 産前6週間前までに「産前産後休業申請書」を提出
3. 母子手帳のコピーを添付
4. 出産後、出生証明書を提出

## 育児休業

### 取得期間

**原則**: 子が1歳になるまで
**延長**: 保育所に入所できない等の事情がある場合、最長2歳まで

### 対象者

- 男女とも取得可能
- 配偶者が育休中でも取得可能
- 有期契約社員も一定条件で取得可

### パパ・ママ育休プラス

両親ともに育休を取得する場合、子が1歳2ヶ月になるまで延長可能

### 出生時育児休業(産後パパ育休)

**対象**: 男性従業員
**取得期間**: 子の出生後8週間以内に4週間まで
**分割**: 2回に分割して取得可能

### 給付金

**育児休業給付金**(雇用保険から支給):
- 最初の6ヶ月: 休業開始時賃金の67%
- 6ヶ月以降: 休業開始時賃金の50%

**社会保険料**: 休業期間中は免除

### 申請方法

1. 育休開始予定日の1ヶ月前までに「育児休業申請書」を提出
2. 母子手帳のコピー、配偶者の育休取得状況がわかる書類を添付
3. 人事部が雇用保険の手続きを代行

### 復職

1. 復職予定日の1ヶ月前までに「復職届」を提出
2. 人事部と復職面談を実施
3. 必要に応じて短時間勤務制度を利用

## 介護休業

### 取得期間

**対象家族1人につき**: 通算93日まで
**分割**: 3回まで分割取得可能

### 対象家族

配偶者、父母、子、配偶者の父母、祖父母、兄弟姉妹、孫

### 要介護状態

負傷、疾病、身体上・精神上の障害により、2週間以上の期間にわたり常時介護を必要とする状態

### 給付金

**介護休業給付金**(雇用保険から支給):
- 休業開始時賃金の67%

### 申請方法

1. 介護休業開始予定日の2週間前までに「介護休業申請書」を提出
2. 要介護状態を証明する書類(診断書、介護保険証等)を添付
3. 人事部が雇用保険の手続きを代行

### 介護短時間勤務

介護休業とは別に、介護のための短時間勤務制度も利用可能

**期間**: 利用開始から3年間
**勤務パターン**:
- 1日6時間勤務
- 週4日勤務
- その他、柔軟な勤務形態

## 申請方法

### システムでの申請(原則)

**アクセス**: 社内ポータル → 勤怠管理 → 各種休暇申請

**基本手順**:
1. ログイン後、該当する休暇種類を選択
2. 必要事項(日付、理由等)を入力
3. 添付書類がある場合はアップロード
4. 「申請」ボタンをクリック
5. 上長がシステム上で承認
6. 承認完了メールが届く

### 緊急時の申請

**病気・家族の不幸等で事前申請が困難な場合**:
1. 始業時刻までに上長に電話・メールで連絡
2. 復帰後、速やかにシステムで正式申請
3. 必要書類を添付

### 承認フロー

1. **一次承認**: 直属上司
2. **二次承認**: 部門長(5日以上の連続休暇の場合)
3. **最終承認**: 人事部(産休・育休・介護休業の場合)

### 申請の取消・変更

**取消**:
- システムで「申請取消」を選択
- 承認前であれば即座に取消可能
- 承認後は上長に連絡の上、取消申請

**変更**:
- 一旦取消し、再度申請
- 緊急の場合は上長に直接連絡

## よくある質問

### Q1: 有給休暇は理由を詳しく書く必要がありますか?

**A**: いいえ。「私用のため」「リフレッシュのため」等、簡潔で構いません。プライベートな理由を詳細に記載する必要はありません。

### Q2: 有給休暇の取得を拒否されることはありますか?

**A**: 原則としてありません。ただし、事業の正常な運営を妨げる場合(繁忙期の複数名同時取得等)は、時季変更権の行使により別の日への変更をお願いすることがあります。

### Q3: 病気で突然休む場合、診断書は必ず必要ですか?

**A**: 3日未満の場合は不要です。3日以上連続する場合は診断書の提出をお願いします。

### Q4: 有給休暇の残日数はどこで確認できますか?

**A**: 社内ポータルの「勤怠管理」→「有給休暇残日数照会」で確認できます。毎月の給与明細にも記載されています。

### Q5: 育児休業中も社会保険料を払う必要がありますか?

**A**: いいえ。育児休業中は社会保険料(健康保険・厚生年金)が免除されます。ただし、将来の年金額には影響しません。

### Q6: 有給休暇を使い切った後、病気で休む場合はどうなりますか?

**A**: 欠勤扱いとなり、給与は支給されません(ノーワーク・ノーペイ)。長期療養が必要な場合は休職制度の利用を検討してください。

### Q7: 年次有給休暇と特別休暇は同時に取得できますか?

**A**: はい。例えば夏季休暇3日と年次有給休暇2日を組み合わせて5連休にすることも可能です。

### Q8: 育児休業から復帰後、すぐにフルタイム勤務が難しい場合は?

**A**: 短時間勤務制度が利用できます。子が小学校3年生修了まで、1日6時間勤務が可能です。
